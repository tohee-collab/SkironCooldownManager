# SkironCooldownManager

SkironCooldownManager expands upon the Blizzard's Cooldown Manager into a more flexible tracking and layout system. It provides support for specialization-based layouts, anchors and the addition of custom timers. Along with Cast & Resource Bars.

## What Is It?

- Reorganizes Blizzard's cooldown viewers into your own anchor layout.
- Lets you style cooldown icons, borders, fonts, swipes, glow behavior, and buff bars.
- Adds custom icons for spells, items, equipment slots, and cast-based timers.
- Supports specialization-specific layouts as well as shared global anchors.
- Includes optional resource bar and cast bar modules that can be anchored into the same setup.

### Essentials, Utility, Buffs & Buff Bars.

- Deconstructs the Blizzard Cooldown Managers and provides you with the most amount of customization.
    - You can assign any of the spells within the Cooldown Managers to any number of anchors, with full control on sizing/placement.

### Custom Icons and Tracking

- Add custom entries for:
  - Spells
  - Items
  - Equipment Slots
  - Timers
- Configure per-icon behavior such as:
  - Hide Off CD.
  - Always Show.
- Custom Icons also can have multiple load conditions, such as Race/Class/Role.

### Visual Styling

- Custom Border Size & Border Colours.
- Customizable Fonts.
- Customizable Cooldown Swipe Colours & Fonts.
- Custom Glows.
- Customizable Pandemic Glows.
- Buff Bar Styling.

### Resource & Cast Bars.

- Advanced Anchoring System.
- Fully customizable.
- Full support for all Power Types (including Tip of the Spear & similar).

## Configuration Overview
